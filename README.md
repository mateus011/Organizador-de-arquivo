# Organizador-de-arquivo
#descrição 
projeto de um organizador de arquivos altomatico, que classifica jpg,pdf e word , com opção de atualizar tipos de arquivos como videos e documentos entre outros

#funcionalidade
-ele utiliza as bibliotecas, os e shutil , para interação com o sistema operacional e movimentação de arquivos
-verificando o tipo 
-verifica se tem duplicatas
